# My Reads Project

To save a little time with this project we were provided with a static template that contained the CSS and HTML we could use. We were tasked with refactoring this template to add interactivity.

## Launching the Application

### Downloading instructions

1. Download the repository
2. Navigate to its location and unzip it
3. Navigate to the location where you unzipped it
4. Open a terminal window and type `npm install`
5. Once that has finished type `npm start` this will launch the app in your browser
6. If the app has not opened in your browser navigate to `localhost:3000`

### Clone instructions

1. Clone the repository
2. Navigate to it's location
3. Open a terminal window and type `npm install`
4. Once that has finished type `npm start` this should open the app in your browser
5. If the app has not opened in your browser navigate to `localhost:3000`
